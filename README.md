# Task Manager API
> Made with Django (Django Rest Framework) 

## Install dependencies

```bash
pip install -r requirements.txt
```

## Create super user (admin)

```bashThe
./manage.py createsuperuser
```

## Run server

```bash
./manage.py runserver
```
